# 📡 NetworkMapper

An Android application for real-time signal strength monitoring using Google Maps.

---

## 🚀 Features

* 📶 Dynamic signal strength measurement (dBm)
* 📍 Location-based tracking
* 🗺️ Google Maps integration
* 🔄 Real-time signal updates while moving

---

## 🧠 How It Works

The app continuously:

1. Captures signal strength (dBm) using TelephonyManager
2. Fetches user location using GPS
3. Displays real-time signal strength on the map

---

## 📊 Signal Representation

* Signal strength is displayed in dBm (decibel-milliwatts)
* Lower values indicate weaker signal (e.g., -110 dBm)
* Higher values indicate stronger signal (e.g., -80 dBm)
